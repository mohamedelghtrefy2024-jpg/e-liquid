# EPP Calculator — موقع حاسبة خلطات السحبة

نسخة "موقع" مستقلة من حاسبة EPP (Phase 1)، شغالة بالكامل على المتصفح (Frontend فقط)
بدون أي حاجة لسيرفر أو قاعدة بيانات. مبنية بـ React + Vite.

---

## 1) التشغيل محلياً (للتطوير)

يحتاج Node.js (الإصدار 18 أو أحدث) مُثبّت على جهازك.

```bash
npm install
npm run dev
```

هيفتح على `http://localhost:5173` تقريباً.

---

## 2) البناء للنشر

```bash
npm run build
```

ده هيعمل فولدر `dist/` فيه الموقع جاهز كملفات HTML/CSS/JS ثابتة (static).
الفولدر `dist/` المرفق في هذه النسخة جاهز بالفعل ومبني — تقدر تستخدمه على طول.

---

## 3) النشر (مجاناً) — اختار طريقة

### أسهل طريقة: Netlify Drop (بدون أي حساب)
1. روح على https://app.netlify.com/drop
2. اسحب فولدر `dist/` كامل وحطه في الصفحة
3. هيديك لينك مباشر للموقع شغال فوراً

### Vercel (لو عايز ربط مع GitHub لتحديثات مستقبلية)
1. اعمل ريبو جديد على GitHub وارفع المشروع ده كامل (ما عدا `node_modules` و `dist`)
2. على https://vercel.com اعمل "New Project" واختار الريبو
3. Vercel هيكتشف إنه Vite تلقائياً (Build command: `npm run build`, Output: `dist`)
4. Deploy — وهيديك دومين مجاني (`*.vercel.app`)

### Netlify عن طريق GitHub (تحديثات تلقائية)
نفس فكرة Vercel: اربط الريبو من https://app.netlify.com — هيبني وينشر تلقائياً مع كل تحديث.

### دومين خاص
أي من الطريقتين فوق (Vercel/Netlify) تقدر بعد كده تضيف دومين خاص بيك (Custom Domain) من إعدادات المشروع مجاناً، وتشتري الدومين من أي مزوّد (Namecheap, GoDaddy...).
